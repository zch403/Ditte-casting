To run front-end:
    ng serve

To run back-end:
    dotnet run

To do new migration:
    dotnet ef migrations add SomeNameForMigration
    dotnet ef database update

To roll back to OldAndWantedMigration do:
    dotnet ef database update OldAndWantedMigration
    dotnet ef migrations remove
